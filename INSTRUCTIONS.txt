TO RUN:

1. in terminal 1 (backend server runs on port 5000):
cd backend

npm install express cors (required dependencies)

export OPENROUTER_API_KEY="the key"
node server.js

-----------------------------------------------------

2. in terminal 2 (frontend server runs on port 3000):
cd meme-frontend
npm start

-----------------------------------------------------


HOW BACKEND WORKS:

1. user inputs prompt
2. memeInput.jsx sends to server.js
3. (STEP 1) server.js calls security-check endpoint
	runs "security_check.py"
	"security_check.py" returns result (either safe or blocked)
4. (STEP 2) server.js calls find-image endpoint
	runs "find_image.py"
	"find_image.py" returns result (filename of image)
